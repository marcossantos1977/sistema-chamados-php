# Sistema Interno de Chamados em PHP

Projeto de estudo para aprender PHP back-end do zero.

A ideia é construir um sistema simples de chamados internos para empresas.

## Objetivo

Aprender PHP, MySQL, organização de código e GitHub criando um projeto realista.

## Funcionalidades planejadas

- Login de usuários
- Cadastro de usuários
- Abertura de chamados
- Listagem de chamados
- Alteração de status
- Resposta aos chamados
- Dashboard administrativo

## Tecnologias

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Git
- GitHub
